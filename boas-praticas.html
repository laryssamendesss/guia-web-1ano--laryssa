Boas Práticas

Semântica & Acessibilidade

  • Use tags semânticas (landmarks) para estruturar conteúdo.
  • Adicione alt descritivo em imagens.
  • Conecte 
<label for="email">Email</label>
<input id="email" type="email" />

Responsividade

  • Adote mobile-first e media queries.
  • Use Flexbox e Grid para layouts.
  • Use unidades relativas (rem, %).
.container{display:grid;grid-template-columns:1fr}
@media(min-width:700px){.container{grid-template-columns:1fr 2fr}}

Organização de CSS

  • Use convenções (BEM) e variáveis CSS.
  • Separe componentes e utilitários.
  • Comente decisões importantes.
/* Ex.: bloco BEM */
.header__title{font-size:1.2rem}

Performance & Versionamento

  • Otimize imagens e use lazy loading.
  • Evite JS desnecessário e carregue scripts deferidos.
  • Commits claros e branches para features.
<img src="foto.jpg" loading="lazy" alt="Descrição" />
